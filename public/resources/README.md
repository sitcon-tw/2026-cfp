# SITCON 2026 資源檔案

此目錄存放 SITCON 2026 活動的宣傳素材資源。

## 需要的檔案

請將以下檔案放置於此目錄：

### 主視覺圖片

1. **visual-square.png** - 主視覺標準版
   - 尺寸：1200×1200 px
   - 格式：PNG
   - 用途：一般社群媒體貼文與網站使用

2. **visual-facebook.png** - Facebook 封面版
   - 尺寸：820×312 px
   - 格式：PNG
   - 用途：Facebook 粉絲專頁封面圖片

3. **visual-youtube.png** - YouTube 封面版
   - 尺寸：2560×1440 px
   - 格式：PNG
   - 用途：YouTube 頻道橫幅圖片

### 頭像與 Logo

4. **avatar.png** - 活動頭像
   - 尺寸：1080×1080 px (1:1 比例)
   - 格式：PNG
   - 用途：個人社群媒體頭像使用

5. **theme-logo.svg** - 年會主題 Logo
   - 格式：SVG (建議同時提供 PNG 版本)
   - 內容：Jam the Chaos 主題標誌
   - 用途：各種尺寸的宣傳素材

## 檔案命名規範

- 檔名使用小寫英文與連字號
- 避免使用空格或特殊字元
- PNG 檔案建議使用 24-bit 或 32-bit (含透明度)
- SVG 檔案請確保路徑已優化

## 使用授權

這些素材僅供 SITCON 2026 活動宣傳使用，請勿用於商業目的。
詳細使用規範請參考網站上的說明。
